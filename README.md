# Grade Five

**Grade Five** is a trusted Japanese vehicle import marketplace for UK buyers. Post your request, compare verified dealer quotes, and track sourcing through UK delivery.

## Stack

- [Next.js](https://nextjs.org) (App Router)
- TypeScript, Tailwind CSS, shadcn/ui
- Demo auth (cookie-based) and in-memory mock data for local MVP

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

| Command        | Description              |
| -------------- | ------------------------ |
| `npm run dev`  | Start development server |
| `npm run build`| Production build         |
| `npm run start`| Run production server    |
| `npm run lint` | ESLint                   |
